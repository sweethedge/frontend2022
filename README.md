## 절대경로 / 상대경로
+ 절대경로
  + 파일의 속성 탭에서 보이는 경로. 이걸 쓸 일이 없고 쓰면 안된다.
+ 상대경로
  + 현재 작성하고 있는 파일을 기준으로 하는 경로. VS Code에서는 dot /를 찍으면 선택지를 보여준다. 동일한 폴더에서는 찍을 필요 없다.
